🏠 PropBot Kya Hai — Simple Language Mein

PropBot ek AI WhatsApp bot hai jo real estate agents ke liye kaam karta hai. Jab koi buyer WhatsApp pe message karta hai — PropBot automatically:

  • ✅ Unka naam, budget, location, BHK poochta hai
  • ✅ Matching properties show karta hai
  • ✅ Site visit book karta hai
  • ✅ Lead score karta hai (Hot/Warm/Cold)
  • ✅ Agent ko alert bhejta hai agar lead hot hai
  • ✅ Sab kuch Google Sheets mein save karta hai

Agent ko kuch nahi karna — bot sab handle karta hai, 24 ghante, 7 din.

⚠️ Yeh yaad rakho: Tum yeh bot real estate agents ko bechte ho. Tumhara customer = Agent/Broker. Unka customer = Property buyer. Tum agent ki problem solve kar rahe ho, buyer ki nahi.

WhatsApp

WhatsApp Setup — 2 Options

Option A — Demo Ke Liye (Abhi)

Twilio Sandbox — Free hai, abhi kaam karta hai

⚠️ Problem: Buyer ko pehle "join [word]" bhejni padti hai sandbox join karne ke liye. Real business ke liye nahi.

Use karo: Demos ke liye, pehle clients ke liye proof of concept

Option B — Production Ke Liye

Gupshup API — Official WhatsApp, Indian company

✅ Real WhatsApp number milta hai. Koi "join" nahi. Buyers directly message kar sakte hain.

Cost: ₹2,999/month + ₹0.40/message

Apply: gupshup.io pe register karo

Strategy: Pehle 2-3 clients Twilio Sandbox se lo (free demo). Unka paisa aaye toh Gupshup lo. Sab clients ko proper WhatsApp de do.
Client Setup

Naye Client Ko Kaise Setup Karo

Client ne paisa diya — ab kya karo. 24 ghante mein deliver karo.

1

Property List Maango

Excel ya WhatsApp pe bhejo. Naam, location, price, BHK, amenities — sab chahiye.

2

Properties.js Update Karo

config/properties.js file mein unki properties add karo. Format exactly same rakhna:

properties.js{ id:1, name:"Property Name", location:"City, Area", price:"₹XX Lakh", area:"XXXX sqft", bhk:"2BHK", status:"Ready to Move", budget_category:"50L_1Cr", amenities:"Gym, Pool, Club", builder:"Builder Name", possession:"Immediate" }

Budget categories: under_50L / 50L_1Cr / 1Cr_2Cr / above_2Cr

3

Agent Phone Number Add Karo

.env file mein: AGENT_PHONE=+91XXXXXXXXXX — yeh number pe hot lead alert aayega.

4

Live Test Karo

Khud WhatsApp pe "Hi" bhejo, poora conversation karo, ek visit book karo. Confirm karo sab kuch chal raha hai.

5

Client Ko Handover Karo

Dashboard URL do, WhatsApp number do, ek 30-min call karo — unhe dikhao kaise use karna hai.

Objections

Objection — Kya Jawab Do

❌ "Bahut expensive hai"

"[Naam] ji, expensive nahi — investment hai. Calculator mein aapne dekha — 3 deals × ₹75K = ₹2.25 lakh. Bot cost ₹15K. Sirf EK extra deal = bot ka pura kharcha recover. Baaki ₹2.1 lakh pure profit. Kya ₹15K invest karna make sense karta hai?"

⚠️ "Soch ke batata hoon"

"Bilkul! Sochna chahiye. Ek kaam karo — 7 din free trial mein shuru karo. Agar 7 din mein 5 qualified leads nahi aaye — full refund. Zero risk. Haan ya naa?"

❓ "Technical nahi hoon, setup mushkil hoga"

"Aapko ek bhi line code nahi likhni. Aap sirf property list aur WhatsApp number dena. Main 24 ghante mein live kar deta hoon. Phir ek call mein sab explain kar deta hoon. Tumhara kaam sirf WhatsApp check karna hai."

❓ "WhatsApp Business se kaam chal jaata hai"

"WhatsApp Business mein aap manually reply karte ho — sirf jab jaag rahe ho. Raat 2 baje lead aaya? Kal subah reply karoge — woh tab tak competitor se deal kar chuka hoga. PropBot 3 seconds mein reply karta hai — 24 ghante. Aur lead qualify bhi karta hai. WhatsApp Business sirf messaging hai — PropBot complete sales system hai."
